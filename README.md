# Homebrew Tap

Homebrew formulae for [jbabin91](https://github.com/jbabin91) packages. Formulas are automatically managed by [cargo-dist](https://opensource.axo.dev/cargo-dist/).

## Install

```sh
brew install jbabin91/tap/autocomplete-rs
```

## Packages

| Package | Description |
|---------|-------------|
| [autocomplete-rs](https://github.com/jbabin91/autocomplete-rs) | A fast, lightweight terminal autocomplete engine |
